# Adult_Income
Analyze features related to adult income using multiple machine learning models.

The data comes from Kaggle's [Adult Income Dataset](https://www.kaggle.com/wenruliu/adult-income-dataset/version/2).

The presentation uses these notebooks:
- [Income Prediction with Census Data](https://www.kaggle.com/geoffreygeo/income-prediction-with-census-data) by Geoffrey
- [Simple Neural Networks using keras](https://www.kaggle.com/ritvikkhanna09/simple-neural-networks-using-keras) by Ritvik.

The purpose of this project is to present technical information for Springboard's Data Science Career Track.
